# Digital Literacy Fundamentals

A modern and responsive e-learning web application designed to educate users about Fake News and Disinformation. The platform provides interactive learning modules, quizzes, and educational resources to improve digital literacy.

## Features

- Responsive modern UI
- Interactive learning modules
- Progress tracking
- Accordion-based learning content
- Video lesson section
- Interactive quiz
- Learning goals
- Mobile-friendly design
- Clean dashboard layout

## Module 1: Fake News & Disinformation

This module covers:

- What is Fake News?
- How misinformation spreads on social media
- Examples of fake news
- Reliable vs unreliable sources
- Fact-checking basics
- Interactive quiz

## Tech Stack

- React.js
- Tailwind CSS
- Vite
- JavaScript
- Lucide React Icons

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
